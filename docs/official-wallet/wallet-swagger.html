Wallet API
 1.3.0 
OAS 3.1

/wallet/openapi

REST API for accessing Wallet financial data. Provides endpoints for financial records, accounts, budgets, and other financial entities for the Wallet application.


Query Filter Prefixes
• certain query parameters require inputs with filter prefixes
• Range filters: gt. gte. lt. lte. eq. — comparisons (>, ≥, <, ≤, =)
• Text filters:
    eq. — exact match
    contains. — substring (case-sensitive)
    contains-i. — substring (case-insensitive)
• see the API Reference for more details and examples

▪️Returns paginated list of financial transaction records.
▪️Use recordDate filter with range operators to scope results. If omitted, a default 3-month window is applied (see appliedRecordDateFilters in the response). Provide any single bound (e.g. gte.2020-01-01) to override.
▪️ID-based lookups (?id=) bypass the default window entirely.
